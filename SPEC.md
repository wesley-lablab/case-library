# 案例库管理系统 - 规格说明

## 1. 项目概述

**项目名称**: 案例库管理系统
**项目类型**: 单页应用 (Vue 3 + TypeScript)
**核心功能**: 通过时间、类型、金额三个维度进行案例筛选和搜索

## 2. 技术栈

- **框架**: Vue 3 + Composition API
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **状态管理**: Pinia
- **构建工具**: Vite

## 3. 页面结构

```
src/
├── components/
│   ├── Navbar.vue           # 顶部导航栏
│   ├── FilterSection.vue     # 筛选区域
│   ├── CaseCard.vue         # 案例卡片
│   ├── CaseList.vue         # 案例列表
│   ├── Pagination.vue       # 分页器
│   └── EmptyState.vue       # 空状态
├── composables/
│   └── useFilter.ts         # 筛选逻辑
├── stores/
│   └── caseStore.ts         # 案例状态管理
├── types/
│   └── index.ts             # 类型定义
├── App.vue
└── main.ts
```

## 4. 视觉规范

### 4.1 颜色系统
```css
:root {
    --color-primary: #1a1a2e;      /* 深蓝黑 */
    --color-secondary: #16213e;    /* 深蓝 */
    --color-accent: #e94560;       /* 活力红 */
    --color-background: #f8f9fa;   /* 温暖白 */
    --color-text: #2d3436;        /* 深灰 */
    --color-muted: #dfe6e9;       /* 浅灰 */
}
```

### 4.2 字体
- **标题**: Noto Serif SC (Google Fonts)
- **正文**: Noto Sans SC
- **数字**: DM Mono

### 4.3 间距
- 基础单位: 4px
- 卡片圆角: 12px
- 阴影: 多层柔和阴影

### 4.4 动画
- 卡片加载: 淡入上移, stagger 80ms
- Hover: 上浮 + 阴影加深, 300ms
- 筛选切换: 200ms ease-in-out

## 5. 功能清单

### 5.1 导航栏
- Logo 和标题
- 统计数字（总案例数、筛选结果数）
- 新增案例按钮

### 5.2 筛选区域
- **日期范围选择器**: 开始日期 + 结束日期
- **类型选择器**: 多选按钮组（民事、刑事、行政、知识产权、合同纠纷、婚姻家庭）
- **金额范围滑块**: 双滑块，支持预设档位
- **筛选标签**: 显示已激活的筛选条件，可移除
- **清除按钮**: 一键清除所有筛选

### 5.3 案例列表
- 响应式网格布局（桌面3列，平板2列，手机1列）
- 案例卡片展示
- 空状态提示
- 分页器

### 5.4 案例卡片
- 案例标题
- 案例编号
- 案例类型标签
- 涉及金额
- 发生日期
- 摘要描述
- Hover 效果

## 6. 响应式断点

| 设备 | 宽度 | 列数 |
|------|------|------|
| 桌面 | ≥1024px | 3列 |
| 平板 | 768-1023px | 2列 |
| 手机 | <768px | 1列 |

## 7. 交互规范

### 7.1 筛选交互
- 日期选择: 点击日期输入框选择
- 类型选择: 点击按钮切换选中状态
- 金额滑块: 拖动滑块，实时更新显示值
- 筛选标签: 点击 × 移除该筛选条件

### 7.2 卡片交互
- Hover: 上浮 4px + 阴影加深
- 点击: 预留详情页入口（当前版本显示提示）

### 7.3 分页交互
- 点击页码: 切换到对应页面
- 点击上一页/下一页: 翻页
- 首页/末页: 到达边界时按钮禁用

## 8. 数据模拟

使用 JavaScript 对象数组模拟案例数据，包含以下字段：
- `id`: 案例ID
- `title`: 案例标题
- `type`: 案例类型
- `amount`: 涉及金额
- `date`: 发生日期
- `summary`: 案例摘要
- `status`: 案例状态
