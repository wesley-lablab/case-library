# 案例库管理系统 - 产品需求文档

## 1. 概念与愿景

一个现代化的案例库管理平台，采用"编辑杂志"风格设计，将数据管理转化为优雅的浏览体验。通过时间、类型、金额三个维度进行精准筛选，让用户能够快速定位所需案例。整体设计追求专业感与设计感的平衡，界面如同翻阅一本精心排版的商业杂志。

## 2. 设计语言

### 美学方向
- **风格定位**：编辑杂志风格 (Editorial/Magazine)
- **核心感受**：专业、精致、清晰、高效
- **视觉特征**：大量留白、优雅排版、清晰层次、精致细节

### 配色方案
- **主色调**：#1a1a2e (深邃蓝黑)
- **次要色**：#16213e (深蓝)
- **强调色**：#e94560 (活力红)
- **背景色**：#f8f9fa (温暖白)
- **文字色**：#2d3436 (深灰)
- **辅助色**：#dfe6e9 (浅灰)

### 字体选择
- **标题字体**：Noto Serif SC (优雅衬线体，体现专业感)
- **正文字体**：Noto Sans SC (清晰无衬线体，保证可读性)
- **数字字体**：DM Mono (等宽数字，用于金额展示)

### 空间系统
- **基础单位**：8px
- **间距节奏**：8, 16, 24, 32, 48, 64px
- **卡片圆角**：12px
- **阴影层次**：多层柔和阴影，营造卡片漂浮感

### 动效哲学
- **页面加载**：卡片依次淡入上移，stagger 间隔 80ms
- **交互反馈**：hover 时轻微上浮 + 阴影加深
- **筛选切换**：列表平滑过渡，高度变化使用弹性缓动
- **悬停效果**：微妙的缩放和阴影变化

### 视觉节奏
- 导航栏：固定顶部，高度 72px，深色背景
- 筛选区：白色卡片，柔和阴影，padding 32px
- 列表区：响应式网格，桌面 3 列，平板 2 列，手机 1 列
- 卡片间距：24px

### 响应式策略
- **桌面端 (≥1200px)**：3列网格，完整筛选器
- **平板端 (768-1199px)**：2列网格，筛选器紧凑排列
- **移动端 (<768px)**：单列堆叠，筛选器可折叠

## 4. 功能与交互

### 核心功能

#### 4.1 时间筛选器
- **交互方式**：日期范围选择器
- **预设选项**：最近7天、最近30天、本月、上季度、自定义
- **视觉表现**：日历图标 + 日期范围文字显示
- **交互细节**：
  - 点击展开日历面板
  - 选择开始和结束日期
  - 选中日期范围高亮显示
  - 清除按钮快速重置

#### 4.2 类型选择器
- **交互方式**：下拉多选菜单 + 标签快速切换
- **类型选项**：
  - 民事案例
  - 刑事案例
  - 行政案例
  - 知识产权
  - 合同纠纷
  - 婚姻家庭
- **视觉表现**：
  - 未选中：描边样式
  - 选中：填充强调色
  - hover：轻微背景变化

#### 4.3 金额范围筛选
- **交互方式**：双滑块范围选择器
- **金额范围**：0 - 10,000,000 元
- **预设档位**：0-10万、10-50万、50-100万、100万以上
- **视觉表现**：
  - 滑块轨道带渐变色
  - 实时显示当前范围
  - 拖动时显示气泡数值

#### 4.4 筛选标签管理
- **显示逻辑**：已激活的筛选条件显示为可移除标签
- **批量操作**：全部清除按钮
- **动画效果**：标签添加/移除使用淡入淡出

### 案例卡片功能

#### 4.5 案例卡片展示
- **信息展示**：
  - 案例标题（突出显示）
  - 案例编号
  - 案例类型（标签）
  - 涉及金额
  - 发生日期
  - 摘要描述
- **交互状态**：
  - default：基础样式
  - hover：上浮 4px + 阴影加深
  - 点击：展开详情/进入详情页

#### 4.6 搜索结果
- **无结果状态**：友好提示 + 清除筛选建议
- **加载状态**：骨架屏占位
- **结果计数**：显示当前筛选结果数量

## 5. 组件清单

### 5.1 导航栏组件 (Navbar)
- **外观**：深色背景，白色文字
- **状态**：
  - default：标准显示
  - scrolled：添加底部阴影
- **内容**：标题 + 统计数字 + 操作按钮

### 5.2 筛选器卡片 (FilterCard)
- **外观**：白色背景，12px圆角，柔和阴影
- **布局**：flex横向排列，响应式堆叠
- **状态**：
  - active：筛选条件激活时显示强调色
  - collapsed：移动端可折叠

### 5.3 日期选择器 (DateRangePicker)
- **外观**：输入框样式 + 日历图标
- **状态**：
  - empty：显示占位文字
  - filled：显示选中日期范围
  - open：展开日历面板

### 5.4 多选下拉 (MultiSelect)
- **外观**：带下拉箭头的输入框
- **状态**：
  - closed：显示选中数量或全部
  - open：展开选项列表
  - hover：背景色变化

### 5.5 金额滑块 (RangeSlider)
- **外观**：双轨道 + 双滑块手柄
- **状态**：
  - dragging：显示数值气泡
  - idle：隐藏气泡
- **显示**：当前范围数值

### 5.6 案例卡片 (CaseCard)
- **外观**：白色背景，圆角，精巧阴影
- **布局**：标题区 + 内容区 + 底部元信息
- **状态**：
  - default：标准样式
  - hover：上浮 + 阴影加深
  - loading：骨架屏

### 5.7 分页器 (Pagination)
- **外观**：居中显示，数字按钮 + 箭头
- **状态**：
  - current：当前页高亮
  - disabled：箭头禁用状态

### 5.8 空状态 (EmptyState)
- **外观**：图标 + 提示文字 + 操作按钮
- **使用场景**：无筛选结果时

## 6. 技术方案

### 前端技术栈
- **框架**：纯 HTML5 + CSS3 + Vanilla JavaScript
- **样式方案**：CSS变量 + Flexbox/Grid 布局
- **动画库**：原生 CSS 动画 + Transitions
- **图标**：内联 SVG 图标

### 数据模拟
- 使用 JavaScript 对象模拟案例数据
- 支持前端筛选、排序、分页
- 模拟异步加载效果

### 浏览器支持
- Chrome 80+
- Firefox 75+
- Safari 13+
- Edge 80+

### 性能目标
- 首屏加载 < 2s
- 筛选响应 < 100ms
- 动画流畅 60fps
