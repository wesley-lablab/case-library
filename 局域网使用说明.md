# 局域网数据共享使用说明

## 🎉 现在不需要手动导入导出了！

我们添加了数据共享功能，局域网内所有电脑可以共享同一套数据。

---

## 📋 使用步骤

### 第一步：启动服务器（在有数据的电脑上）

1. 关闭之前的命令行窗口
2. **双击 `启动服务器.bat`**（不是原来的运行.bat）
3. 等待服务器启动完成
4. 看到提示后，先访问 `http://localhost:5173/migrate.html` 迁移数据

### 第二步：迁移数据

1. 在服务器电脑的浏览器中打开：`http://localhost:5173/migrate.html`
2. 点击 **"从浏览器导出数据"**
3. 点击 **"复制到剪贴板"**
4. 然后点击下方的输入框，粘贴刚才复制的数据
5. 点击 **"导入到服务器"**
6. 看到成功提示后，数据就迁移完成了！

### 第三步：其他电脑访问

1. 在服务器的命令行窗口中，找到显示的局域网地址，例如：
   ```
   局域网访问: http://192.168.1.100:5173
   ```
2. 其他电脑在浏览器中输入这个地址
3. 登录后就可以看到共享的数据了！

---

## 📝 数据存储

现在数据保存在服务器电脑的 `data.json` 文件中，所有电脑都共享这一个文件。

---

## ⚠️ 注意事项

1. **服务器电脑必须保持开机**，并且运行 `启动服务器.bat`
2. 所有电脑必须在**同一局域网**内
3. 如果服务器电脑的IP地址变了，其他电脑需要使用新的IP地址访问
4. 修改数据后会自动保存，所有电脑刷新页面就能看到更新
